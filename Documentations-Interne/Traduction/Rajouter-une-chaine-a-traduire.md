# Rajouter une traduction
Pour rajouter une traduction il faut rajouter la chaine dans **locales/glpi.pot**.

## Exemple
```pot
	msgid "Menu position"
	msgstr ""
```

# Information
Une fois merger sur le depot main. La traduction sera disponible sur le Weblate.
